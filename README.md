# YouTube 統計トラッカー (クライアントサイド実装)

YouTube動画の再生数、高評価数、コメント数をリアルタイムで監視・グラフ化するウェブアプリケーション。複数の動画を同時に追跡し、増加率も表示します。

## 機能

- YouTube動画URLまたは動画IDから統計情報を取得
- 複数の動画を同時にトラッキング
- 再生回数、高評価数、コメント数のリアルタイム監視
- 取得したデータをグラフで視覚化
- 24時間・週間の増加率表示
- YouTubeライクなカウンターアニメーション
- 10秒ごとのリアルタイム自動更新

## 必要条件

- YouTube Data API v3のAPIキー
- 任意のウェブサーバー(またはローカルファイルでも動作可能)

## セットアップ

1. 以下のファイルをウェブサーバーに配置するか、ローカルでHTMLを開きます：
   - public/index.html
   - public/css/style.css
   - public/js/chart.js
   - public/js/counter-animation.js
   - public/js/main.js

2. ブラウザでindex.htmlを開きます

3. 画面上部のフォームにYouTube APIキーを入力します

## YouTube API Keyの取得方法

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. プロジェクトを作成
3. YouTube Data API v3を有効化
4. 認証情報からAPIキーを作成
5. **重要**: プロダクション環境では、APIキーに適切な制限(HTTPリファラーなど)を設定してください

## 使用方法

1. アプリ画面でYouTube APIキーを入力して「APIキーを設定」をクリック
2. YouTube動画のURLまたは動画ID（例: `dQw4w9WgXcQ`）を入力
3. 「追加」ボタンをクリックして動画をトラッキングリストに追加
4. 複数の動画を追加可能で、リストから選択して表示を切り替えられます
5. 動画の統計情報とグラフが表示されます
6. 「トラッキング停止」ボタンで個別の動画の監視を終了できます
7. 24時間・週間の増加率は、十分なデータが収集された後に表示されます

## 注意事項

- このクライアントサイド実装では、APIキーがブラウザに露出するため、個人的な使用または教育目的にのみ使用してください
- 本番環境では、APIキーを保護するためにバックエンド実装を検討してください
- トラッキングデータはローカルストレージに保存されるため、ブラウザのデータをクリアすると失われます
